# Dijkstra Visualizer

Quick Vite + React visualization of Dijkstra's algorithm on a grid.

Getting started (Windows):

1. Install dependencies:

```powershell
npm install
```

2. Start dev server:

```powershell
npm run dev
```

Open the address shown by Vite (usually http://localhost:5173).

Controls:
- Click cells to toggle walls.
- "Visualize Dijkstra" runs the animation.
- "Random Walls" fills random walls.
